Object Choices

Each button, when clicked calls a function in the object-choices.js file.

The simple object button calls the function within an object named simpleObject. This function returns and prints out the first and last name of author, which are stored within the object, to the console and on the html page.

The function object button calls the sayName function of the function object, which must first be initialized, then returns and prints "function object" to the console and html page.

The custom class button calls the sayName function within the class object, which is initialized and constructed to store the strings "Custom" and "Class" as the first and last name, and then returns the variables and prints them to the console and html page.
